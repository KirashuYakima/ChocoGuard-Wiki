Welcome to ChocoGuard-Privacy Policy!

What is this Privacy Policy for?

The bot may use stored data. No usage of data outside of the aformentioned cases will happen and the data is not shared with any 3rd-party site or service.

Stored Information.

The bot may store the following information automatically when being invited to a new Discord Server:

    What information would get stored:
  • Guild-ID
  • Server Name

With this information a message would get sent to our logging channel saying that this bot is invited in guild X with the name X.

Updating or Editing Data.

The data may be updated or edited when using specific commands.

Updating or Editing data will require the input of an end user, and data that can be seen as sensitive, such as content of a message, may need to be stored when using certain commands.

No other actions may update or edit the stored information at any given time.

    What we are saving if using specific commands:
  • Ticket System Setups.
    • Ticket ID, Catagory-ID, Channel-IDs, Ticket Messages, Guild-ID and User-IDs.

  • Suggestion System Setups.
    • Guild-ID, Channel-IDs and User-IDs.

  • Anti-Swear System Setups.
    • Guild-ID.

  • Blacklisting System.
    • Guild-ID if blacklisted Guild, User-IDs and Reasons.

  • Economy System, account creation and balance.
    • Guild-ID, User-ID and Balance. (Cash + Bank)

  • Slowmode System, saves time on channels who have slowmodes enabled by using our bot.
    • Guild-ID, Time, Reason and User-ID.

  • TimeOut System, saves time on members by timeouting them using our bot.
    • Guild-ID, Time, Reason and User-ID.

Removal of Data.

Manual removal

Manual Remove is possible by joining our Support Server with /support

For security reasons please do mind that you need to give us proof that you own the server.

Only a server owner may request manual removal of data and requesting it will result in the bot being removed from the server, if still present on it.

What data can be removed if the bot stays?

  • Ticket System Setup

  • Suggestion System Setup

  • Anti-Swear System Setup
